# polyscope-tests

A repository for testing [Polyscope](https://polyscope.dev) workflows — parallel coding agents, branch management, and collaborative development.

## Purpose

This repo serves as a sandbox for experimenting with Polyscope features:

- Running multiple agents in parallel across isolated workspaces
- Testing branch creation, commits, and pull request workflows
- Validating context sharing between agents via `.context` directories
- Exploring how agents handle various project structures and languages

## Getting Started

Clone the repository:

```sh
git clone https://github.com/sschlein/polyscope-tests.git
cd polyscope-tests
```

Then open the project in Polyscope to spin up workspaces and start testing.
